Malt is a web application that enables users to place food orders from restaurants through
an online system. The program provides convenient ordering services to customers and at the same time provides an effective restaurant and order management system.


## Features

- User authentication and authorization
- Restaurant listings and menu browsing
- Add restaurants to favorites
- Search for foods and restaurants
- Sort food by price
- User reviews and ratings
- Food ordering and order tracking
- Order updates via email and notifications
- Secure payment gateway integration (Stripe, eSewa, and Khalti)
- Create and update restaurant details
- Restaurant dashboard with income, orders, menu item metrics, graphs, and charts
- Restaurant order and menu management system
- Host and manage restaurant event details


## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Java, Spring Boot

**DBMS:** PostgreSQL
